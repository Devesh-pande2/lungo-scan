## About This Project

Lungo-Scan PRO is a signal processing system that analyzes cough sounds to detect tuberculosis using a deterministic sigmoid logistic equation derived from MATLAB-based STFT analysis.

---

## Repository Files

| File | Description |
|------|-------------|
| `lungo_scan.html` | Web-based clinical interface |
| `lungo_scan_v8_6.m` | MATLAB source code |
| `PROJECT_DOCUMENTATION.pdf` | Complete technical documentation |
| `positive tb patient.wav` | TB-positive cough sample |
| `negetive tb patient.wav` | Healthy cough sample |
| `Doc/` | MATLAB result images |

---

## How to Use

1. Open `PROJECT_DOCUMENTATION.pdf` for complete methodology
2. Run `lungo_scan_v8_6.m` in MATLAB to see validation results
3. Open `lungo_scan.html` in browser to test web interface
